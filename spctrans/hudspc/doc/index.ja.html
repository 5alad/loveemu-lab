Hudson SPC2MIDI

Hudson SPC2MIDI（hudspc）は、ハドソンのシーケンスに対応したMIDIコンバータとその関連資料です。

サウンドプログラムについて

MMLっぽいデザインです。Ver 1.xx と Ver 2.xx が存在します。下記は現時点で判明している使用タイトルです。

  • No Version String
    • Super Bomberman 2 (1.xx 互換)
    • Hagane (1.xx 互換)
    • An American Tail: Fievel Goes West (1.xx と 2.xx の中間?)
  • Version 1.xx
    • Super Bomberman 3 (1.16)
    • Super Genjin 2 (1.16E)
    • Caravan Shooting Collection (1.17s)
  • Version 2.xx
    • Do-Re-Mi Fantasy (2.10)
    • Tengai Makyou Zero (2.27b)
    • Super Bomberman 4 (2.28)
    • Kishin Douji Zenki 3 (2.28)
    • Same Game (2.30a)
    • Super Bomberman 5 (2.32)
    • Bomberman B-Daman (2.32)

つかいかた

hudspc --helpで起動するとシンタックスとか出てくるので、テキトーに使ってみてください。

Q&A

ツッコミ入れられそうな部分に先回りして答えるコーナー。

一括変換できないの？
バッチファイルやシェルスクリプトで対処してください。
hudspc.cssってどこにあるんですか？
docディレクトリにあります。適宜、base.cssとセットでコピーして使ってください。
全然違う曲に変換されてしまいます
適当に--songというオプションを併用してみるとたぶん変換できます。スーパーボンバーマン2とか。
Why don't you write a English manual?
Because it's a bit too difficult for me, also I'm extremely lazy.
