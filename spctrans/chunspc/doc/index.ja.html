Chunsoft summer/winter SPC2MIDI

Chunsoft summer/winter SPC2MIDI（chunspc）は、チュンソフトのサウンドプログラムに対応したMIDIコンバータとその関連資料です。

サウンドプログラムについて

変換対象のサウンドプログラムは、弟切草の作曲者として知られる三俣千代子さんが制作したものです。使用タイトルは以下の通りです。

  • 弟切草 (Otogirisou)
  • ドラゴンクエストV 天空の花嫁 (Dragon Quest 5)
  • トルネコの大冒険 不思議のダンジョン (Torneco no Daibouken: Fushigi no Dungeon)
  • かまいたちの夜 (Kamaitachi no Yoru)

ROM/ARAMを覗くと中にsummerやwinterという文字列を見つけることができるため、私はこのプログラムをsummer/winterと呼んでいます。弟切草で使われたのがsummer、後にそれをバージョンアップしたものがwinterです。

Tool converts unexpected song, why?
ChunSPC will try searching current song location by some RAM conditions, however it is not always correct. Otogirisou is the most problematic because it can have multiple songs in RAM. Also, Otorigisou SPC collection in SNESmusic.org is dumped during loading a song, it really confuses the tool. You can "redump" those SPCs by using SNES SPC700 Player.
Can I redump Otogirisou SPC without changing the song start timing then? 既存の弟切草SPCの再生タイミングを変えることなく再ダンプすることは可能？
Yes, it can be done by the following steps.
  1. Open the target SPC by a hex editor, then edit offset $109c (6f → ff) and save it.
  2. Open the edited SPC by SNES SPC700 Player, I guess you will not hear any sound. (If the song plays correctly, probably redump will not help you. Use --song option to tell the expected song index to ChunSPC instead.)
  3. Save SPC snapshot and open it by a hex editor, then edit offset $109c (ff → 6f) and save it.
  4. Done. Try converting it by ChunSPC.
